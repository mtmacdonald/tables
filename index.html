Tables jQuery plugin (version x.x.x)

Tables is a jQuery plugin for dynamic HTML data tables on the client side.

Features

  • AJAX data loading in JSON format
  • Configurable column widths
  • Pagination
  • Forward / back buttons
  • Search filter

Tables requires a server backend for data access. A sample PHP script and reference JSON format is provided.
